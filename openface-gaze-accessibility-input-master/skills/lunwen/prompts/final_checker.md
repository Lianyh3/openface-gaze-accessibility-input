# Final Checker Prompt

交付前必须逐项检查：

1. 章节完整
2. 字数接近样文目标
3. 参考文献比例符合要求
4. 图表编号连续
5. 是否残留占位符
6. `.docx` 是否真实存在
7. 摘要 / Abstract / 参考文献 / 致谢样式是否正确
8. 英文字体是否正确
9. 图表是否真实插入
10. 如果使用模板：封面是否保留
11. 如果使用模板：目录是否保留或成功生成
12. 如果使用模板：正文是否从正确位置开始替换
13. 如果使用模板：`中文摘要`、`ABSTRACT`、`目　　录` 是否误带 `第N章`
14. 如果使用模板：是否只存在一个目录
15. 如果使用模板：正文标题是否存在编号叠加，如 `第1章第1章`、`1.11.1`
16. 如果使用模板：`参考文献`、`致　　谢`、`声　　明`、附录是否误带 `第N章`
17. 如果使用模板：两个字的后置标题是否保持模板字间空格
18. 如果使用模板：后置标题和附录标题是否被脚本错误覆盖为默认字号/字体
19. 图题、表题是否为单倍行距
20. 图片是否超出版心或与正文发生覆盖
21. 如果输入源是 `.doc`：是否已转换为 `.docx`
22. 如果生成了 Word 目录字段：是否已设置打开文档时刷新域
