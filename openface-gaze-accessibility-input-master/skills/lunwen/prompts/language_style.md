# Language Style Prompt

用于统一论文语言风格。

默认目标：

- 语言学术化、克制、稳定，不口语化
- 叙述以第三人称或客观陈述为主，避免“我认为”“我们觉得”
- 不堆砌空话，不用明显 AI 套话
- 一段只表达一个中心意思，避免句子过长
- 系统型论文强调“需求-设计-实现-测试”链路
- 科研型论文强调“问题-方法-实验-分析”链路

写作时必须注意：

1. 章节内部逻辑递进清晰
2. 段首句点题，段末句收束
3. 图表前有引导语，图表后有分析语
4. 不凭空扩写项目中不存在的模块
5. 避免连续使用相同句式，如“首先…其次…最后…”泛滥
6. 摘要语言应更浓缩，结论语言应更归纳

如果用户给了样文，必须同时提取：

- 常见句式
- 常见过渡方式
- 章节展开节奏
- 图表分析的表达习惯

然后优先贴近样文风格，而不是使用统一模板腔。
