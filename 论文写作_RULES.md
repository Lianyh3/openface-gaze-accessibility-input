# 论文写作规则（集美大学本科论文适配）

适用对象：`论文初稿_*.md` 全部章节草稿  
当前题目：`基于OpenFace的实时视线估计优化`

## 1. 结构规则

1. 第1章必须采用模板口径：`1.1研究意义`、`1.2研究内容`、`1.3研究方法`，并保留`1.4`与`1.5`。  
2. 目录最多列到三级标题；若正文出现四级内容，用段内编号`（1）（2）`表达。  
3. 每章必须有“本章小结”，且小结不得重复摘要表述。  
4. 标题长度尽量控制在15字以内，避免口语化标题。

## 2. 论证规则

1. 章节必须遵循“问题—方法—结果/作用—局限”逻辑。  
2. 关键贡献章节（第3、4、5章）允许篇幅明显更长，不要求平均分配字数。  
3. 避免“功能清单式”流水账，重点描述机制、因果关系和设计权衡。  
4. 不夸大结论；样本规模有限时必须显式写“适用边界”。

## 3. 一致性规则

1. 题目主线必须统一为“实时视线估计优化”，眼控系统定位为“验证平台”。  
2. 第3章方法变量和名词在第4、5章必须保持同名同义。  
3. 第5章实验结论不得与第3章方法目标冲突。  
4. 若出现“失败案例”，必须解释失败发生层（命中/触发/提交）而非笼统归因。

## 4. 公式与编号规则

1. 公式统一按章编号，格式示例：`（3.1）`、`（5.2）`。  
2. 正文引用统一写法：`由式（3.1）可见`。  
3. 同章公式编号连续，不跳号、不重复。  
4. 不用截图公式，全部使用可编辑公式表达。

## 5. 文献规则

1. 第1章现状与第2章技术基础必须有文献支撑。  
2. 正文引用采用上标编号（最终排版时处理为上标）。  
3. 文末参考文献不少于15篇，英文不少于3篇。  
4. 若正文存在`[n]`占位，定稿前必须替换为真实可查文献条目。

## 6. 实验规则（第5章）

1. 指标必须包含：准确性、稳定性、实时性、输入任务效果。  
2. 同时给“成功样例”和“失败样例”分析，避免单一正向结果。  
3. 实验数据需可追溯到日志或报告文件。  
4. 对比实验需说明比较口径一致性（输入、参数、阈值、评价标准）。

## 7. 表达与排版预检查

1. 章节叙述不用“我觉得/我做了”口语表达。  
2. 禁止“代码很强/技术很牛”等主观评价。  
3. 中文正文采用学术陈述句，避免过长并列句。  
4. 插图和表格必须在正文先被引用再出现。

## 8. 提交前检查清单

1. 目录与正文标题完全一致。  
2. 公式编号、图表编号、引用编号全部连续。  
3. 第1章与第5章结论口径一致，不自相矛盾。  
4. 文献条目与正文引用一一对应。  
5. 全文满足学校字数与格式要求后再导入模板排版。

## 9. 文献落地流程（必须执行）

1. 先确定第1章占位编号与真实文献的映射关系（例如 `[1]...[9]`）。  
2. 再将真实文献写入“参考文献”章节，按正文出现顺序编号。  
3. 对每条文献至少检查：作者、题名、年份、会议/期刊、可检索链接。  
4. 若使用占位编号，定稿前必须完成替换，不允许保留空占位。  
5. 优先引用原始论文或官方文档，避免二手转载与不明来源摘要站点。
